# Getting Started with Create React App

### Clone
git clone https://github.com/Canulo21/TAC-system.git

### install depedencies
npm install

npm install -g npm@10.5.1

npm install -g yarn

npm install -g concurrently

npm install react-scripts --save

npm install -g nodemon

### Run build
yarn dev
